# Admin Client

Admin panel cho ADO Fine Jewelry.

## Cài đặt

```bash
npm install
```

## Chạy Development Server

```bash
npm run dev
```

Admin panel sẽ chạy tại: http://localhost:5174

## Build Production

```bash
npm run build
```

## API Configuration

Backend API mặc định: `http://localhost:5000`

Có thể cấu hình qua biến môi trường `VITE_API_URL` trong file `.env`

## Chức năng

- Quản lý Đá Quý (Gemstones)
- Quản lý Trang Sức (Jewelry)
- Quản lý Bộ sưu tập (Collections)
- Quản lý Danh mục
- Quản lý Hero Slides
- Quản lý Tin tức/Blog
- Cấu hình hệ thống (Cloudinary, Logo, Sections)
